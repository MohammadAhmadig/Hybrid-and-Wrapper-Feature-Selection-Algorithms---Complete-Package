
IGIS+
====
Written by Songyot Nakariyakul, 2018

If you find this MATLAB program useful, please cite

S. Nakariyakul, A hybrid gene selection algorithm based on interaction information for microarray-based cancer classification, PLOS One (2018) submitted.


To use IGIS+, please run demo.m 

The IGIS+ algorithm requires functions from MIToolbox. Users are required to download and install them from the following website. 

MIToolbox: http://www.cs.man.ac.uk/~pococka4/MIToolbox.html


Microarray datasets in MATLAB format required for our work can be downloaded from http://songyot.ece.engr.tu.ac.th/software.html
